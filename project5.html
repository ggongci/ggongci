TuneMatch (음악 취향 추천 시스템)

이 프로젝트는 개인화 음악 추천 알고리즘 개발을 목표로 합니다.

사용자가 약 10일간 청취한 음악 데이터를 수집하여, 사용자의 선호 장르를 분석하고 비슷한 곡들을 추천하는 추천 시스템 알고리즘을 구현할 예정입니다.

주요 기능 (예정):

  • 🎧 청취 기록 데이터 수집
  • 📊 장르/아티스트 통계 분석
  • 🔍 KNN 또는 유사도 기반 추천 알고리즘 적용
  • 🎼 새로운 곡 추천 리스트 제공
  • 🌐 간단한 웹 인터페이스 구현 가능

사용 예정 기술: Python, Pandas, Scikit-learn, Matplotlib, Flask

예정 시작: 2~3학년 추천 알고리즘 및 AI 기초 학습 단계
