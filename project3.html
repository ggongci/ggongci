Orange3 단순선형회귀 분석

이 프로젝트에서는 Orange3라는 GUI 기반 데이터 분석 도구를 활용하여 단순선형회귀 분석을 수행했습니다.

Orange3는 파이썬 기반의 오픈소스 머신러닝, 시각화 플랫폼으로, 복잡한 코딩 없이 데이터 마이닝과 통계 분석이 가능합니다.

수행 내용:

  • 📊 광고비 vs 매출액 데이터 수집
  • 📈 단순선형회귀 분석 수행
  • 📉 모델 적합성 확인 (R² 값, p-value 확인)
  • 📊 시각적 그래프 분석 및 해석
  • 📝 분석 결과를 보고서로 작성 및 발표

사용 도구: Orange3, 단순선형회귀, 시각화 위젯

진행 시기: 2025년 1학기 (빅데이터분석및시각화 수업)
